# Homework: Semi-Supervised Learning

## Задача

Данная задача моделирует проблему semi-supervised learning, где доступны как размеченные, так и неразмеченные данные.

## Что нужно сделать

1. Запустите baseline (`baseline/train_baseline.ipynb`) и убедитесь, что все работает корректно.

2. Реализуйте решение, которое улучшит метрику **F1 macro** на тестовой выборке.

3. **Основная цель**: достичь **F1 macro ≥ 0.32** (32%) на тестовой выборке.

## Данные

- `train_labeled.csv` - размеченные данные для обучения
- `train_unlabeled.csv` - неразмеченные данные
- `test.csv` - тестовая выборка

## Baseline

Baseline использует только размеченные данные и достигает F1 macro ~0.27 на тестовой выборке.
